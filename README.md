# Rumaan Moqthar - Portfolio

A responsive Single Page Application (SPA) portfolio built with **React** and **Vite**.

## Features
- **Component-Based Architecture**: Modular React components for Home, About, Projects, and Contact.
- **Smooth Scrolling**: Simple navigation between sections.
- **Responsive Design**: optimized for mobile and desktop.

## Project Structure
```
src/
├── components/       # Individual section components (Navbar, Home, etc.)
├── App.jsx          # Main application layout
├── index.css        # Global styles
└── main.jsx         # Entry point
public/
└── images/          # Static assets
```

## How to Run
1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```
3.  Build for production:
    ```bash
    npm run build
    ```
